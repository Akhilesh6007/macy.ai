## 🌟 Overview

MACY.AI is an all-in-one AI platform that allows users to access multiple AI models from a single interface.

Instead of switching between different AI services, MACY.AI provides a unified experience for:

* 💬 AI Chat
* 🌐 Website Generation
* 🎨 AI Image Creation
* 💻 Code Generation
* 🤖 Workflow Automation
* ⚡ API Access

---

## 🚀 Installation

```bash
# Clone repository
git clone https://github.com/yourusername/macy-ai.git

# Navigate into project
cd macy-ai

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## ⚙️ Environment Variables

Create a `.env.local` file:

```env
MONGODB_URI=
JWT_SECRET=
OPENAI_API_KEY=
GEMINI_API_KEY=
CLAUDE_API_KEY=
DEEPSEEK_API_KEY=
```
